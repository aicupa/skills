# Todolist Skills

Claude Code skills for the [Todolist](https://github.com/aicupa) desktop app. Provides AI-assisted knowledge for `.todo` file format, plugin development, and plugin view communication.

## Install

```bash
npx skills add https://github.com/aicupa/skills
```

## Skills

| Skill                | Description                                                                         |
| -------------------- | ----------------------------------------------------------------------------------- |
| `todolist-knowledge` | .todo file format, plugin system, plugin API, view protocol, and plugin contributes |
